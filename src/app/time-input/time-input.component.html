ngx-time-input

A custom control that guesses a time from user input and normalizes it to 24-hour HH:mm format. It strives to be forgiving about the input format and locale-aware.

API

The component implements `ControlValueAccessor`. The model is a string in the format `HH:mm` (24-hour time). - `selector: 'ngx-time-input'` ##### Inputs - `displayFormat: string` The format for displaying the date in the input. See Moment's [formatting docs](https://momentjs.com/docs/#/displaying/format/). Default: `'LT'` (time in the locale). - `inputPlaceholder: string` A string to be used as the input's placeholder attribute. Default: `'Enter a time'` - `inputId: string` A string to be used as the input's id. - `inputClass: string` A string to be used as the input's class. Pass any error classes here.

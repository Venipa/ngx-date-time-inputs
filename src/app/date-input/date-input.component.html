ngx-date-input

A custom control that guesses a date from user input and normalizes it to YYYY-MM-DD format. It strives to be forgiving about the input format and locale-aware.

API

The component implements `ControlValueAccessor`. The model is a string in the format `YYYY-MM-DD`. - `selector: 'ngx-date-input'` ##### Inputs - `displayFormat: string` The format for displaying the date in the input. See Moment's [formatting docs](https://momentjs.com/docs/#/displaying/format/). Default: `'LL'` (e.g., September 4, 1986). - `inputPlaceholder: string` A string to be used as the input's placeholder attribute. Default: `'Enter a date'` - `inputId: string` A string to be used as the input's id. - `inputClass: string` A string to be used as the input's class. Pass any error classes here.
